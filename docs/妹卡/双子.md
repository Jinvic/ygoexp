# 双子

一卡link4确实香，但一卡动对应的就是一卡停。不过双子补点还是有些的所以也能接受，还是能玩的。主轴够小，大量卡位能投入大量手坑之类的用来干扰护航，又称手坑均。

主轴小但额外就很吃卡位了，本家吃掉七个卡位终端也不够强，剩下卡位的取舍就很重要。之前投入了【链接弟子】【连接信徒】的ip套，虽然能加速ip双穹，但这套首先手卡要胡，动点补点全上手。其次电子界展开链又臭又长吃坑还没法妥协，不是很喜欢还是去掉了。本家link4很好用但是用少一个，意味着双子最多三回合解决对手再就转不动了，要是场地能回收就好了。另一个常和双子绑定的终端是【破械双王神 来迎】，因为和大蓝很契合。但其实个人用着一般，有资源出来迎对面也没什么反抗能力了，来迎没卡炸。想出来迎时要么被对面打断要么自己资源不够。

主卡扔一堆手坑解场玩多了也腻，所以有时也会投入些别的玩玩。由于大蓝的炸卡，可以投入合适的古遗物卡。小红小蓝分别是光暗属性，所以可以投入一些光暗支援，刨掉没用的墓地资源。比如深渊兽系列，以及【混沌奈芙提斯】的取对象除外。

25-12-12更新：

两张双子新卡强化了初动，阿宝阿库进一步扩充了场值。但双子本家并没有合格的终端，所以现在的双子基本都是混刻魔玩的。姬丝基勒的link怪都是光恶魔，可以很方便转到刻魔做出大怒涛或震怒作为阻抗。但是刻魔是没办法转到双子系统的。而且刻魔也是以扩充场值为主的思路，但双子做阿宝阿库的路线需要清空场值，也就是只能后置刻魔系统来做阻抗，没办法前置骗坑护航。总之，在刻魔双子这套卡组中反而不期望刻魔内容先手开而是通过双子去调度刻魔。现在的双子能够通过爆场值一波做出一个能赢的场子，但也就只有这一波了。被解后只剩剩一些主播的复活和刻魔的回收之类，有时会陷入空有场值但没终端可出的尴尬境地。

也不知道还有没有不带刻魔的纯双子构筑存在，有空可以找找。刻魔大人还是太权威了。

参考combo：通招小红拉小蓝，小红变交易堆甜蜜，甜蜜自跳。交易变小棺材拉红泪堆铁匠，两小蓝变大蓝拉交易，红泪和交易变破械神王拿红狗。大蓝和圣王做阿宝阿库，丢一手苏生小蓝拉霜精，做大红拉大蓝再做麻烦桑妮。桑妮解题拉大红大蓝抽一，两怪拼大棺材融落泪，落泪拿铁匠回手。铁匠找踢腿，自跳和落泪做大怒涛。踢腿找路里丢路里自跳，红狗炸路里起跳，墓地神王苏生桑妮。最后红狗和大棺材做蓝狗。EP阿宝阿库回场。终场大怒涛两次特招效果康，阿宝阿库和麻烦桑妮两炸，蓝狗一次加速链接，做小叶还有小叶的一次除外和一时除外。手上阿宝丢一加大红两次抽一还剩五卡。如果做场过程中有任意补点，最后可以不做蓝狗而是做狮鹫封怪效。
